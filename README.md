# Interior Design Landing Page

Responsive landing page for an interior design studio.

## Technologies

* HTML5
* SCSS
* JavaScript

## Features

* Responsive layout (mobile / tablet / desktop)
* Modern UI design
* Smooth scrolling navigation
* Image gallery / portfolio section
* Burger menu for mobile devices

## Sections

* Hero
* About
* Services
* Our Team
* Portfolio
* Contact form
* Minimalist Room
